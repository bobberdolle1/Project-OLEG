# Changelog

Все заметные изменения в проекте "Олег" будут задокументированы в этом файле.

## [3.0.0] - 2025-11-27

### Добавлено
- Система "Plug & Play" - автоматическая настройка при добавлении в чат
- Умное приветствие пользователей с подтверждением "Я не бот"
- Модуль зрения - анализ изображений с помощью мультимодальных ИИ
- Система "Мозг Олега" (RAG) - контекстное обучение и память
- Рандомные ответы - бот сам вставляет реплики в чат
- Администрирование через личные сообщения
- Векторная база данных ChromaDB для RAG
- Мультимодальная модель Ollama для анализа изображений
- Команда `/admin` для доступа к админ-панели в ЛС
- Команда `/random_toggle` для управления рандомными ответами
- Команда `/vision_test` для проверки модуля зрения
- Система модерации через inline-меню
- Обновленная система управления админами
- Черный список с локальными и глобальными банами

### Изменено
- Удалена веб-панель управления
- Обновлен системный промпт для лучшей адаптации к контексту
- Улучшена система модерации с тремя режимами: "Лайт", "Норма", "Диктатура"
- Обновлена система приветствия участников
- Обновлена документация для отражения новых возможностей
- Улучшена защита от промпт-инъекций
- Улучшена система отслеживания токсичности

### Удалено
- Веб-админка (заменена на систему управления через ЛС)
- Устаревшие модули и зависимости

### Исправлено
- Исправлены ошибки в системе модерации
- Исправлены проблемы с обработкой изображений
- Исправлены ошибки в системе сохранения контекста
- Исправлены проблемы с многопоточностью в асинхронных функциях
